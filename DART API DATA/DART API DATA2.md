## 공통 정리 메모

- **분석 우선순위**
  1. 순이익률
  2. ROE
  3. 총자산영업이익률
  4. 매출총이익률 / 판관비율

- **버릴 지표**
  - 자본금영업이익률
  - 납입자본이익률  
  → 자본금이 작은 한국 기업 특성상 왜곡 심함

- DART 수익성지표 API는
  - 지표 누락이 잦고
  - 기업 간 비교 시 **정의 불일치** 발생 가능  
  → 핵심 지표는 **직접 계산** 권장
