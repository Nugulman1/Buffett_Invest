# DART API(fnlttSinglAcnt)로 FCF·ROIC·WACC 자동 계산 가능 여부 검증

## 1. API로 가져올 수 있는 데이터 (dart_2024_financial_data.json 기준)

`fnlttSinglAcnt` 응답은 **재무상태표(BS)** + **손익계산서(IS)** 만 포함한다. **현금흐름표(CF)는 없음.**

### 1.1 재무상태표 (BS)

| account_nm | sj_div | 비고 |
|------------|--------|------|
| 비유동자산 | BS | |
| 자산총계 | BS | |
| 유동부채 | BS | |
| 비유동부채 | BS | |
| 부채총계 | BS | |
| 자본금 | BS | |
| 이익잉여금 | BS | |
| 자본총계 | BS | |

- **유동자산** 미포함 (샘플 기준).
- **현금및현금성자산** 없음.
- **단기차입금, 장기차입금, 사채, 리스부채** 등 이자부채 세부 없음 (유동부채·비유동부채 총계만 존재).

### 1.2 손익계산서 (IS)

| account_nm | sj_div | 비고 |
|------------|--------|------|
| 매출액 | IS | |
| 영업이익 | IS | |
| 법인세차감전 순이익 | IS | |
| 당기순이익(손실) | IS | |
| 총포괄손익 | IS | |

- **이자비용** 없음.

### 1.3 현금흐름표 (CF)

- **제공 안 함.** `sj_div`에 CF 없음.
- 따라서 **영업활동현금흐름(CFO)**, **유형자산 취득**, **무형자산 취득** 등 **모든 CF 계정 부재.**

## 2. FCF / ROIC / WACC 계산에 필요한 항목 (calculator.py 기준)

| 지표 | 필요 필드 | DART 제공 |
|------|-----------|-----------|
| **FCF** | `cfo` (영업활동현금흐름) | ❌ CF 없음 |
| | `tangible_asset_acquisition` (유형자산 취득) | ❌ CF 없음 |
| | `intangible_asset_acquisition` (무형자산 취득) | ❌ CF 없음 |
| **ROIC** | `operating_income` (영업이익) | ✅ IS |
| | `equity` (자기자본) | ⚠️ 자본총계로 대체 가능 |
| | `interest_bearing_debt` (이자부채) | ❌ 세부 없음 |
| | `cash_and_cash_equivalents` (현금및현금성자산) | ❌ BS 없음 |
| **WACC** | `equity` | ⚠️ 자본총계로 대체 가능 |
| | `interest_bearing_debt` | ❌ 세부 없음 |
| | `interest_expense` (이자비용) | ❌ IS 없음 |

## 3. 다중회사 주요계정.md / 주요재무지표.md

- **다중회사 주요계정**: BS·IS 위주 계정만 기술. 현금흐름표·이자비용·현금및현금성자산·이자부채 세부 미포함.
- **다중회사 주요재무지표**: ROE, 매출총이익률 등 **비율 지표**만 제공. FCF·ROIC·WACC **원시 금액** 또는 **계산용 계정** 미제공.

---

## 4. 결론

**DART API(fnlttSinglAcnt)만으로는 FCF·ROIC·WACC 자동 계산에 필요한 데이터를 채울 수 없다.**

- **현금흐름표(CF)** 자체가 없어 **FCF** 계산 불가.
- **이자비용**, **현금및현금성자산**, **이자부채 세부**가 없어 **ROIC·WACC** 계산 불가.

따라서 **1번(DART API 확장)** 은 **fnlttSinglAcnt** 기준으로는 불가.  
(다른 DART API·엔드포인트에서 위 계정 제공 여부는 별도 확인 필요.)
