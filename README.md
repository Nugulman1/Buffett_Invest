# 투자 필터 백엔드

## 프로젝트 소개

버핏의 투자책을 보고 영감을 받아 장기 투자를 해보고 싶은 마음이 생겨 장기 투자를 위한 기업 선별 기능, 주요 재무지표 수집 기능을 구현한 프로그램입니다.

## 주요 기능

1. 대한민국 상장 기업들(약 2850개)을 필터를 통해 선별하는 기능을 제공합니다.
2. 상장 기업들의 최근 5년 사업보고서, 최근 분기보고서에서 중요 지표들 수치를 제공합니다.
3. 기업의 세부적인 지표들(ROIC, WACC, FCF)을 더 편하게 계산할 수 있는 계산기를 제공합니다.
4. 즐겨찾기 기능, 메모등의 기능을 제공합니다.

## 기술 스택

- Django 6.0.x
- Django REST Framework 3.16.x
- Python 3.12–3.14 (권장: 3.13)
- SQLite, python-dotenv 1.2.x

## 데이터 소스

- DART (OpenDART API)
- ECOS (한국은행 경제통계시스템 API) - 국채 금리
